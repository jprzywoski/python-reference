===========================
e | E (scientific notation)
===========================

Description
-----------
Returns a float multiplied by the specified power of 10.

Syntax
------
[0-9].e(-)?[0-9]

[0-9].E(-)?[0-9]

Example
-------
>>> 1.1
1.1
>>> 1.1e0
1.1          # 1.1 * 10**0
>>> 1.1e1
11.0         # 1.1 * 10**1
>>> 1.1e2
110.0        # 1.1 * 10**2
>>> 1.1e3
1100.0       # 1.1 * 10**3
>>> 8e-2
0.08         # 8 * 10**-2
